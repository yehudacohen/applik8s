# Code Of Conduct

This project is intended to be a respectful, technical, and welcoming space.

Expected behavior:

- Be direct without being hostile.
- Assume good intent while staying precise about technical risk.
- Keep criticism focused on code, design, tests, documentation, or operational behavior.
- Respect maintainers' responsibility to preserve correctness, security, and project scope.

Unacceptable behavior:

- Harassment, threats, or discriminatory language.
- Personal attacks.
- Publishing private security reports before maintainers have had a reasonable chance to respond.
- Pressuring contributors to bypass tests, review, or compatibility rules.

Maintainers may remove comments, close issues, or restrict participation to protect the project and community.
